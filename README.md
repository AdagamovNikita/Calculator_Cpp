# Калькулятор на C++/CLI (Windows Forms)

Настольный калькулятор с графическим интерфейсом, реализованный на C++/CLI с использованием Windows Forms (.NET). Поддерживает базовые арифметические операции и содержит набор встроенных тестов.

---

## Возможности

| Функция          | Описание                                        |
|------------------|-------------------------------------------------|
| `+` `-` `×` `÷` | Сложение, вычитание, умножение, деление         |
| `%`              | Проценты: вычисляет X% от первого числа         |
| `±`              | Смена знака текущего числа                      |
| `,`              | Ввод десятичной запятой                         |
| `AC`             | Полный сброс (All Clear)                        |
| `=`              | Вычисление результата                           |
| `T`              | Запуск встроенных тестов (кнопка в углу)        |
| `✕`              | Выход из приложения                             |

## Структура проекта

```
Calculator_Cpp/
├── src/
│   ├── Calculator.h       # Интерфейс класса логики вычислений
│   ├── Calculator.cpp     # Реализация арифметики (без зависимостей от UI)
│   ├── MainForm.h         # Объявление Windows Forms-формы
│   └── MainForm.cpp       # Реализация UI-формы и обработчиков событий
├── main.cpp               # Точка входа приложения
├── CalculatorEXAM.vcxproj # Файл проекта Visual Studio
├── .gitignore
└── README.md
```

**Ключевой принцип разделения:**  
- `Calculator.h / Calculator.cpp` — чистая C++ логика, не зависит от Windows Forms  
- `MainForm.h / MainForm.cpp` — только UI; делегирует вычисления в `Calculator`  
- `main.cpp` — минимальная точка входа

## Инструкция по запуску

### Через Visual Studio (рекомендуется)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/AdagamovNikita/Calculator_Cpp.git
   ```
2. Откройте файл `CalculatorEXAM.vcxproj` в Visual Studio 2022.
3. В панели инструментов выберите конфигурацию **Debug | x64**.
4. Нажмите **▶ Начать отладку** (`F5`) или **Ctrl+F5** для запуска без отладчика.

### Через командную строку (Developer Command Prompt)

```cmd
cd Calculator_Cpp
msbuild CalculatorEXAM.vcxproj /p:Configuration=Release /p:Platform=x64
x64\Release\CalculatorEXAM.exe
```

## Встроенные тесты

Нажмите кнопку **`T`** (верхний левый угол формы), чтобы запустить 5 автоматических тестов:

| № | Операция             | Пример          | Ожидаемый результат |
|---|----------------------|-----------------|---------------------|
| 1 | Сложение             | 9999 + 100      | 10099               |
| 2 | Умножение            | 9999 × 100      | 999900              |
| 3 | Деление              | 9999 ÷ 100      | 99.99               |
| 4 | Вычитание            | 9999 − 100      | 9899                |
| 5 | Проценты             | 9999 % 100      | 9999                |

Результат отображается в диалоговом окне.
